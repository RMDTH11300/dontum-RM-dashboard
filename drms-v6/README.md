# DRMS Sprint 5 — Risk Register อัตโนมัติ

เพิ่มการสร้าง Risk Register จาก Incident และ Risk Profile ตามหน่วยงานหลักที่แก้ไข พร้อมส่งออก CSV และพิมพ์ PDF

## จุดสำคัญ
- ไม่กระทบ Dashboard V1
- ปีงบประมาณยึดจากชุดข้อมูลโดยตรง
- เลือกหลายหน่วยงานได้
- มาตรการและการติดตามเป็นร่างตั้งต้น ต้องให้หน่วยงานเจ้าภาพทบทวนก่อนใช้จริง


## Sprint 6
- Risk Matrix 5×5 จาก Risk Profile อัตโนมัติ
- แนวโน้มเปรียบเทียบปีงบประมาณ 2566–2569
- Checklist ความพร้อมข้อมูลสำหรับ HA
- สรุปข้อความสำหรับการทบทวนคุณภาพในรายงาน HA


## Sprint 7
- เพิ่มมุมมองสรุปและแบบฟอร์ม HA สำหรับ Risk Profile / Risk Register
- ช่องวัตถุประสงค์ ตัวชี้วัด เป้าหมาย มาตรการเพิ่มเติม ผู้รับผิดชอบ กำหนดเวลา และผลติดตาม แก้ไขได้
- บันทึกร่างในเบราว์เซอร์ของเครื่องผู้ใช้
- แสดงประวัติการบันทึกร่าง
- ส่งออก Word, Excel, CSV และพิมพ์/PDF


## Sprint 8
- เพิ่ม HA Report Generator แบบแก้ไขข้อความได้
- เพิ่มหัวข้อบริบท จุดเด่น ประเด็นพัฒนา แผนพัฒนา และ Checklist หลักฐาน
- บันทึกร่างรายงาน HA ในเบราว์เซอร์
- เก็บประวัติการบันทึก
- ส่งออก Word, Excel, CSV และ PDF


## DRMS V6.1 Final
หน้า Admin เปลี่ยนเป็น Data Center สำหรับ:
- นำเข้า Excel จากระบบเดิม
- ตรวจสอบ Data Quality
- ส่งออก JSON สำหรับ GitHub
- สำรอง/กู้คืนข้อมูล
- ตั้งค่าระบบ
- บันทึกประวัติการนำเข้าและส่งออก

หมายเหตุ: ปีงบประมาณถูกกำหนดจากช่องที่ Admin เลือกโดยตรง ส่วนวันที่ใช้เพื่อตรวจสอบและแจ้งเตือนเท่านั้น ระบบจะไม่เปลี่ยนปีงบประมาณของชุดข้อมูลเอง
