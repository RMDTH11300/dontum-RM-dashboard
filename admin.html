ขั้นตอนอัปเดตข้อมูลขึ้น GitHub

  1. โหลดไฟล์ปีที่ต้องการ แล้วเพิ่มหรือแก้ไขข้อมูล
  2. กด ส่งออก JSON สำหรับ GitHub จะได้ไฟล์ชื่อ incidents_2569.json
  3. เข้า GitHub Repository → โฟลเดอร์ data → Add file → Upload files
  4. อัปโหลดไฟล์ใหม่ทับชื่อเดิม แล้วกด Commit changes
  5. รอ GitHub Pages อัปเดต แล้วเปิด Dashboard กด Ctrl + F5
